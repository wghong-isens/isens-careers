## 아이디어 A: "Blueprint Precision" — 산업 설계도 미학

**Design Movement**: Industrial Blueprint / Technical Drafting

**Core Principles**:
1. 설계도(Blueprint)의 정밀함과 격자 구조를 인터페이스 언어로 사용
2. 딥 네이비 + 사이안 라인의 강렬한 대비로 기술적 신뢰감 표현
3. 모든 요소가 측정 가능하고 의도적인 위치에 배치됨
4. 텍스트와 선(Line)의 조화로 정보 계층 구조 명확화

**Color Philosophy**:
- 배경: 딥 네이비 (#0F172A) — 깊이와 전문성
- 주요 강조: 일렉트릭 블루 (#2563EB) — 에너지와 혁신
- 라인/보조: 사이안 (#38BDF8) — 기술적 정밀함
- 텍스트: 순백 + 연한 회색 — 가독성 최우선

**Layout Paradigm**:
- 비대칭 그리드: 좌측 70% 콘텐츠 + 우측 30% 사이드 패널
- 헤더는 좌측 정렬, 풀블리드 섹션 없이 컨테이너 기반
- 수직 타임라인으로 채용 프로세스 표현

**Signature Elements**:
1. 가는 수평선과 수직선이 배경에 격자를 형성 (설계도 느낌)
2. 섹션 번호가 큰 모노스페이스 폰트로 배경에 희미하게 표시
3. 버튼과 카드에 날카로운 모서리 + 파란 테두리 선

**Interaction Philosophy**:
- 호버 시 라인이 그려지는 애니메이션
- 스크롤에 따라 격자 패턴이 미세하게 이동

**Animation**:
- 진입 시 요소들이 좌측에서 슬라이드인
- 숫자 카운터 애니메이션 (통계 섹션)
- 라인 드로잉 효과 (SVG stroke-dashoffset)

**Typography System**:
- 제목: Space Grotesk Bold (기술적이면서 현대적)
- 본문: Noto Sans KR Regular
- 강조 숫자: JetBrains Mono (모노스페이스)

## 아이디어 B: "Kinetic Manifesto" — 움직이는 선언문 미학 ✅ 선택

**Design Movement**: Swiss International Typographic Style + Motion Design

**Core Principles**:
1. 타이포그래피가 레이아웃의 주인공 — 텍스트 자체가 시각 요소
2. 강렬한 대각선과 사선 분할로 역동성 표현
3. 블루와 화이트의 강한 대비, 최소한의 색상 사용
4. 정보는 명확하게, 비주얼은 대담하게

**Color Philosophy**:
- 배경: 순백 (#FFFFFF) — 깔끔함과 신뢰
- 주요 강조: 아이센스 블루 (#2563EB) — 브랜드 아이덴티티
- 다크 섹션: 슬레이트 (#1E293B) — 무게감과 권위
- 포인트: 라이트 블루 (#DBEAFE) — 부드러운 강조

**Layout Paradigm**:
- 대각선으로 분할된 히어로 섹션 (clip-path polygon)
- 좌우 교차 배치 (홀수 섹션: 텍스트 좌/이미지 우, 짝수 반대)
- 풀블리드 컬러 블록 섹션이 스크롤 흐름을 끊어주는 구조
- 스티키 네비게이션 + 진행 인디케이터

**Signature Elements**:
1. 대형 섹션 타이틀이 배경 텍스처처럼 희미하게 반복
2. 블루 사선 구분선 (45도 각도의 clip-path)
3. 채용 포지션 카드에 좌측 블루 보더 라인

**Interaction Philosophy**:
- 스크롤 연동 패럴랙스 효과
- 카드 호버 시 위로 살짝 떠오르는 효과 (translateY)
- 버튼 호버 시 배경이 좌에서 우로 채워지는 슬라이드 효과

**Animation**:
- 섹션 진입 시 Framer Motion으로 fadeInUp
- 숫자 통계는 카운트업 애니메이션
- 히어로 텍스트는 단어별로 순차 등장

**Typography System**:
- 제목: Pretendard ExtraBold / Black (한국어 최적화, 강렬함)
- 부제목: Pretendard SemiBold
- 본문: Noto Sans KR Regular
- 영문 강조: DM Sans Bold

## 아이디어 C: "Organic Grid" — 유기적 격자 미학

**Design Movement**: Brutalist Warmth / Organic Modernism

**Core Principles**:
1. 불규칙한 카드 크기와 비대칭 배치로 생동감 표현
2. 따뜻한 오프화이트 배경에 강렬한 블루 포인트
3. 두꺼운 테두리와 그림자로 물리적 존재감 부여
4. 텍스처와 노이즈로 디지털 느낌 탈피

**Color Philosophy**:
- 배경: 크림 화이트 (#FAFAF8) — 따뜻함
- 강조: 딥 블루 (#1D4ED8) — 신뢰와 안정
- 보조: 앰버 (#F59E0B) — 에너지와 도전 (포인트)
- 다크: 차콜 (#1F2937) — 텍스트와 구조

**Layout Paradigm**:
- 마소너리(Masonry) 그리드로 채용 포지션 표시
- 대형 타이포그래피 블록이 레이아웃을 지배
- 섹션 간 두꺼운 검정 라인으로 구분

**Signature Elements**:
1. 두꺼운 검정 테두리 카드 (4px border)
2. 강한 박스 섀도우 (offset shadow, 브루탈리스트 스타일)
3. 배경에 미세한 노이즈 텍스처

**Interaction Philosophy**:
- 카드 호버 시 그림자 방향 변화
- 버튼 클릭 시 눌리는 물리적 효과

**Animation**:
- 스크롤 시 요소들이 스태거(stagger) 방식으로 등장
- 배경 노이즈가 미세하게 움직임

**Typography System**:
- 제목: Gmarket Sans Bold (한국어 개성 있는 폰트)
- 본문: Noto Sans KR
- 숫자: Oswald Bold
